# Absolventský Velehrad 2027

Webová stránka pro Absolventský Velehrad — čtyřdenní setkání mladých lidí na poutním místě Velehrad.

**19. – 22. 8. 2027**

## O projektu

Absolventský Velehrad je akce, která se koná každé dva roky a nabízí společné slavení eucharistie, přednášky, diskuze, workshopy i prostor pro ztišení. Web slouží jako hlavní informační zdroj pro účastníky i zájemce.

## Technologie

- HTML, CSS, JavaScript (vanilla, bez frameworků)
- Parallax efekt v hero sekci
- Fotogalerie pomocí [lightGallery](https://www.lightgalleryjs.com/)
- Responzivní design
- Deploy přes GitHub Pages

## Lokální spuštění

Stačí otevřít `index.html` v prohlížeči, nebo použít libovolný lokální server:

```bash
npx serve .
```

## Deploy

Stránka se automaticky nasadí na GitHub Pages při push na větev `main` pomocí GitHub Actions workflow.
